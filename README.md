# SolidUseCase

A flexible UseCase pattern that works *with* your workflow, not against it.

## Installation

Add this line to your application's Gemfile:

    gem 'solid_use_case'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install solid_use_case
